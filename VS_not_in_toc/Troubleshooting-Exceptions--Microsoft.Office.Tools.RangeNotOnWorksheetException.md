# Troubleshooting Exceptions: Microsoft.Office.Tools.RangeNotOnWorksheetException
A <xref:Microsoft.Office.Tools.RangeNotOnWorksheetException?qualifyHint=False> exception is thrown when a view control is created programmatically with a range that is not from the worksheet that corresponds to the controls collection.  
  
## Associated Tips  
 Make sure that the range you specify corresponds to the correct worksheet.
