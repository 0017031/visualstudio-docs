# Troubleshooting Exceptions: System.ComponentModel.Win32Exception
The exception that is thrown for a Win32 error code.  
  
## Remarks  
 Win32 error codes are translated from their numeric representations into a system message when they are displayed. Use the <xref:System.ComponentModel.Win32Exception.NativeErrorCode?qualifyHint=False> property to access the numeric representation of the error code.
