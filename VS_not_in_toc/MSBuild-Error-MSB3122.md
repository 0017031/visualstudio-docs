# MSBuild Error MSB3122
**MSB3122: Use of file associations requires full trust.**  
  
 Publishing an application that configures file associations requires that the application be a full trust application.  
  
### To correct this error  
  
-   Enable ClickOnce security settings and set the application to a full trust application. For more information, see [How to: Enable ClickOnce Security Settings](../VS_IDE/How-to--Enable-ClickOnce-Security-Settings.md).
